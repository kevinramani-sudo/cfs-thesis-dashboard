Interactive dashboard using the same four-variable response logic as the current Python model: panel rationalisation intensity, truss rationalisation intensity, opening/detailing complexity, and prefabrication level.
Scope: CFS framing process cost, CFS framing A1–A5 embodied carbon, and construction process-hours only. This dashboard is a decision-support layer, not a structural design, consent, quantity-surveying, or scheduling substitute.
